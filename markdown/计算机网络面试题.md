**TCP协议与UDP协议的区别？**

| UDP |	TCP |
| ---- | ---- |
|面向无连接                       |面向有链接（三次握手）|
|有单播，多播，广播功能             |仅有单播|
|面向报文                    	|面向字节流|
|不可靠性                        |	可靠传输|
|头部开销小，传输数据报文时是很高效的	|提供拥塞控制/提供全双工通讯|
